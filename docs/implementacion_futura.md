# Implementación Futura

- Módulo de asistencias
- Módulo de pagos y colegiatura

## Reportes y Exportación
- Exportación de datos
- Dashboard estadístico

## Mejoras Operativas y de Experiencia de Usuario (UX)

- Diseño responsivo

## Seguridad
- Seguridad de sesión
- Recuperación de contraseña por correo electrónico (Envío de token temporal mediante SMTP para auto-recuperación por parte del usuario).
